CSS - Folhas de Estilo em Cascata

CSS (Cascading Style Sheets) é usado para estilizar páginas HTML, definindo cores, fontes, layouts e muito mais.

Exemplo de CSS

O exemplo abaixo mostra como estilizar o corpo da página e o cabeçalho:

                /* Estilizando o corpo da página */
                body {
                    font-family: Arial, sans-serif; /* Define a fonte padrão */
                    margin: 0; /* Remove margens padrão */
                    padding: 0; /* Remove padding padrão */
                }

                /* Estilizando o cabeçalho */
                header {
                    background-color: #333; /* Cor de fundo escura */
                    color: #fff; /* Cor do texto branca */
                    padding: 10px; /* Espaçamento interno */
                    text-align: center; /* Centraliza o texto */
                }

No exemplo acima, aplicamos um fundo escuro no cabeçalho com texto branco e removemos margens e padding do corpo da página para ajustar o layout.

Exemplo de Media Query

O exemplo abaixo altera o estilo do cabeçalho em telas menores:

                /* Media Query para telas menores */
                @media (max-width: 600px) {
                    header {
                        font-size: 14px; /* Reduz o tamanho da fonte */
                        padding: 5px; /* Diminui o espaçamento interno */
                    }
                }

Esse código ajusta o tamanho do texto e o espaçamento do cabeçalho para telas pequenas, melhorando a responsividade.

Exemplo de Flexbox

O exemplo abaixo demonstra o uso do Flexbox para criar um layout com elementos centralizados:

                /* Layout com Flexbox */
                .container {
                    display: flex; /* Ativa o modo flexbox */
                    justify-content: center; /* Centraliza os itens horizontalmente */
                    align-items: center; /* Centraliza os itens verticalmente */
                    height: 100vh; /* Altura total da viewport */
                }

                .item {
                    background-color: #4caf50; /* Cor de fundo do item */
                    padding: 20px; /* Espaçamento interno */
                    color: white; /* Cor do texto */
                }

No exemplo acima, uma div com classe container é usada para centralizar um item na tela.

Exemplo de Grid Layout

O exemplo abaixo cria um layout de grade com três colunas:

                /* Layout com Grid */
                .grid-container {
                    display: grid; /* Ativa o Grid Layout */
                    grid-template-columns: repeat(3, 1fr); /* Três colunas de tamanhos iguais */
                    gap: 10px; /* Espaçamento entre os itens */
                }

                .grid-item {
                    background-color: #2196f3; /* Cor de fundo do item */
                    color: white; /* Cor do texto */
                    padding: 20px; /* Espaçamento interno */
                    text-align: center; /* Centraliza o texto */
                }

Esse exemplo cria uma grade com três colunas igualmente espaçadas, usando a propriedade grid-template-columns.
