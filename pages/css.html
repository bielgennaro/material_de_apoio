CSS - Folhas de Estilo em Cascata

O CSS é a cereja do bolo em um site! Ele é responsável por transformar uma página sem graça em algo visualmente atraente. Enquanto o HTML organiza o conteúdo, o CSS define como tudo vai parecer: as cores, o tamanho das fontes, os espaçamentos, e até animações.

Exemplo de CSS

O exemplo abaixo mostra como estilizar o corpo da página e o cabeçalho: 

                /* Estilizando o corpo da página */
                body {
                    font-family: Arial, sans-serif; /* Define a fonte padrão */
                    margin: 0; /* Remove margens padrão */
                    padding: 0; /* Remove padding padrão */
                }

                /* Estilizando o cabeçalho */
                header {
                    background-color: #333; /* Cor de fundo escura */
                    color: #fff; /* Cor do texto branca */
                    padding: 10px; /* Espaçamento interno */
                    text-align: center; /* Centraliza o texto */
                }

No exemplo acima, aplicamos um fundo escuro no cabeçalho com texto branco e removemos margens e padding do corpo da página para ajustar o layout.

Exemplo de Media Query

O exemplo abaixo altera o estilo do cabeçalho em telas menores:

                /* Media Query para telas menores */
                @media (max-width: 600px) {
                    header {
                        font-size: 14px; /* Reduz o tamanho da fonte */
                        padding: 5px; /* Diminui o espaçamento interno */
                    }
                }

Esse código ajusta o tamanho do texto e o espaçamento do cabeçalho para telas pequenas, melhorando a responsividade.

Exemplo de Flexbox

O exemplo abaixo demonstra o uso do Flexbox para criar um layout com elementos centralizados: 

                /* Layout com Flexbox */
                .container {
                    display: flex; /* Ativa o modo flexbox */
                    justify-content: center; /* Centraliza os itens horizontalmente */
                    align-items: center; /* Centraliza os itens verticalmente */
                    height: 100vh; /* Altura total da viewport */
                }

                .item {
                    background-color: #4caf50; /* Cor de fundo do item */
                    padding: 20px; /* Espaçamento interno */
                    color: white; /* Cor do texto */
                }

No exemplo acima, uma div com classe container é usada para centralizar um item na tela.

Exemplo de Grid Layout

O exemplo abaixo cria um layout de grade com três colunas:

                /* Layout com Grid */
                .grid-container {
                    display: grid; /* Ativa o Grid Layout */
                    grid-template-columns: repeat(3, 1fr); /* Três colunas de tamanhos iguais */
                    gap: 10px; /* Espaçamento entre os itens */
                }

                .grid-item {
                    background-color: #2196f3; /* Cor de fundo do item */
                    color: white; /* Cor do texto */
                    padding: 20px; /* Espaçamento interno */
                    text-align: center; /* Centraliza o texto */
                }

Esse exemplo cria uma grade com três colunas igualmente espaçadas, usando a propriedade grid-template-columns.

3 Formas de Linkar o CSS ao HTML

  1. CSS Externo: É o método mais recomendado. Você coloca todo o CSS em um arquivo separado (geralmente com a extensão .css) e o conecta ao HTML com a tag <link>. Isso deixa seu código mais organizado e facilita a manutenção. 
    <link rel="stylesheet" href="styles.css">

    Quando usar? Sempre que o CSS for usado em várias páginas ou for grande o suficiente para justificar um arquivo separado.

  2. CSS Interno: Se for algo específico para uma única página ou uma estilização temporária, você pode usar a tag <style> dentro do <head> do HTML. 
    <style>
    body {
        background-color: lightblue;
    }
</style>

    Quando usar? Para páginas simples ou testes rápidos de estilo.

  3. CSS Inline: Aqui o estilo é aplicado diretamente no elemento HTML usando o atributo style. Embora funcione, esse método deve ser evitado porque dificulta a manutenção. 
    <h1 style="color: red; font-size: 24px;">Título Estilizado</h1>

    Quando usar? Apenas em situações emergenciais ou quando for algo extremamente pontual.

Estilizando Elementos

Vamos começar com um exemplo básico: mudar o fundo da página, definir a fonte e personalizar o cabeçalho. 

body {
    font-family: Arial, sans-serif; /* Escolhe uma fonte moderna e padrão */
    margin: 0; /* Remove as margens automáticas */
    padding: 0; /* Remove os espaçamentos automáticos */
}

header {
    background-color: #333; /* Fundo escuro */
    color: #fff; /* Texto branco */
    text-align: center; /* Centraliza o texto */
    padding: 20px; /* Adiciona espaçamento interno */
}

O que acontece? O site ganha uma aparência mais limpa e moderna, com o cabeçalho em destaque e o fundo ajustado.

Media Queries: Estilo Responsivo

Sites modernos precisam funcionar bem em celulares, tablets e computadores. Com o CSS, você pode criar regras específicas para cada tamanho de tela usando media queries. Olha só: 

@media (max-width: 600px) {
    header {
        font-size: 16px; /* Texto menor para caber na tela */
        padding: 10px; /* Reduz o espaçamento */
    }
}

O que acontece? Em telas menores que 600px, o cabeçalho se adapta, ficando mais compacto e fácil de visualizar.

Flexbox: Controle Absoluto de Layout

O Flexbox é uma ferramenta poderosa para organizar elementos na página. Ele ajuda a centralizar, alinhar e distribuir os itens do jeito que você quiser. 

.container {
    display: flex; /* Ativa o Flexbox */
    justify-content: center; /* Alinha os itens horizontalmente no centro */
    align-items: center; /* Alinha os itens verticalmente no centro */
    height: 100vh; /* Faz o container ocupar toda a altura da tela */
}

.item {
    background-color: #4caf50; /* Fundo verde */
    padding: 20px; /* Espaçamento interno */
    color: white; /* Texto branco */
}

O que acontece? O conteúdo fica centralizado tanto na horizontal quanto na vertical. Ideal para criar designs simétricos e limpos.

Grid Layout: Dividindo Espaços

O Grid é como uma tabela moderna, só que com muito mais controle. Perfeito para criar layouts complexos, como galerias ou painéis. 

.grid-container {
    display: grid; /* Ativa o Grid Layout */
    grid-template-columns: repeat(3, 1fr); /* Cria 3 colunas de tamanhos iguais */
    gap: 10px; /* Espaçamento entre os itens */
}

.grid-item {
    background-color: #2196f3; /* Fundo azul */
    color: white; /* Texto branco */
    padding: 20px; /* Espaçamento interno */
    text-align: center; /* Centraliza o texto */
}

O que acontece? Você cria uma grade flexível com três colunas, onde cada item é espaçado automaticamente. Ideal para organizar elementos visualmente.

Resumo

Com o CSS, você tem o poder de transformar uma página simples em algo bonito, responsivo e funcional. Ele é essencial para qualquer desenvolvedor que queira criar interfaces de qualidade!
