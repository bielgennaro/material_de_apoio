CSS - Folhas de Estilo em Cascata

CSS (Cascading Style Sheets) é usado para estilizar páginas HTML, definindo cores, fontes, layouts e muito mais.

Exemplo de CSS

O exemplo abaixo mostra como estilizar o corpo da página e o cabeçalho:

/* Estilizando o corpo da página */
body {
    font-family: Arial, sans-serif; /* Define a fonte padrão */
    margin: 0; /* Remove margens padrão */
    padding: 0; /* Remove padding padrão */
}

/* Estilizando o cabeçalho */
header {
    background-color: #333; /* Cor de fundo escura */
    color: #fff; /* Cor do texto branca */
    padding: 10px; /* Espaçamento interno */
    text-align: center; /* Centraliza o texto */
}

No exemplo acima, aplicamos um fundo escuro no cabeçalho com texto branco e removemos margens e padding do corpo da página para ajustar o layout.
