HTML - Linguagem de Marcação de Hipertexto

  • (Hypertext Markup Language) é a linguagem padrão para criação de páginas na web;
    • Isto significa que é um código utilizado para descrever a estrutura de um documento, mas não é a verdadeira apresentação dele;
  • O HTML diz ao navegador, tudo sobre o conteúdo recebido, e a estrutura da página;
  • O HTML define cabeçalhos, parágrafos, listas, tabelas entre muitos outros elementos, por meio de marcas (tags).

Instalações e Uso

  • VS Code LINK AQUI;
  • Extensão a ser instalada no VS Code HTML, Javascript;
  • Todas as aulas:
    1. Crie uma pasta com o nome da aula (em um local da sua preferência);
    2. Abra o VS Code;
    3. Clique em Arquivo > Abrir Patas;
    4. Selecione a pasta criada no passo 1 > Selecionar;
    5. No VSCode clique em Arquivo > Novo arquivo;
    6. Crie um arquivo de nome index.html.

Para visualizar sua página web...

  1. No navegador de arquivos, encontre sua pasta;
  2. Clique com o botão direito do mouse sobre o arquivo index.html > Abrir com... > Google Chrome (ou outro navegador web).

A seguir, veremos a estrutura básica de um documento HTML:

Estrutura básica de um documento HTML

O exemplo abaixo mostra a estrutura básica de um documento HTML com suas principais tags:

                <!DOCTYPE html> /* Declara que o documento está utilizando HTML5 */
                <html lang="pt-br"> /* Tag raiz do documento HTML, com o idioma definido */
                <head> /* Cabeçalho do documento */
                    <meta charset="UTF-8"> /* Define a codificação de caracteres */
                    <title>Título da Página</title> /* Título exibido na aba do navegador */
                </head>
                <body> /* Corpo do documento, onde o conteúdo é exibido */
                    <h1>Cabeçalho Principal</h1> /* Cabeçalho ou título principal */
                    <p>Parágrafo de texto.</p> /* Texto em formato de parágrafo */
                </body>
                </html>

Neste exemplo, a tag <html> contém todo o conteúdo do documento. O <head> define metadados e informações sobre o documento, enquanto o <body> contém o conteúdo visível na página.

O que é uma Tag?

Tag é um comando usado em HTML para estruturar o conteúdo de uma página web. Ela é composta por uma tag de abertura e uma tag de fechamento. Entre essas tags, colocamos o conteúdo a ser exibido na página.

Tags HTML

Definições de Tags HTML
Tag HTML Descrição
<html> Define um documento HTML.
<head> Apresenta informações gerais sobre a página e de configuração.
<main> Utilizado para especificar o conteúdo principal, de maior relevância, de um documento.
Cada página deve apresentar apenas um conteúdo principal.
Exemplo: espaço reservado para a apresentação do conteúdo de um blog.
<body> Define o corpo do documento.

Exemplo: cabeçalho de uma página.
<nav> Utilizado para representar um agrupamento de links de navegação.

<aside> Utilizado para criar um conteúdo de apoio, ou adicional, ao conteúdo principal.
Exemplo: artigos com temas relacionados de um blog.
<h1> a <h6> Para indicar que um texto é um título em nossa página, utilizamos a tag de heading;
São tags de conteúdo e vão de <h1> a <h6>.
A variação entre as tags de heading definem o grau de destaque do texto.
<p> Define um parágrafo.
<span> tag para representar um texto sem significado (deve ser usado quando nenhum outro elemento de texto semântico representar um significado adequado).
<br> Insere uma única quebra de linha.
<hr> Define um filete horizontal.
<!--> Define um comentário.
<strike> Permite tachar algum trecho: uma linha reta horizontal fica sobreposta ao texto.
<sub> Possibilita o subscrito: um trecho de texto fica abaixo da linha das demais partes do texto.
<sup> Permite o sobrescrito: um trecho de texto fica acima da linha das demais partes do texto.
<big> e <small> Modifica o tamanho de um trecho do texto, tornando-o um pouco maior (big) ou menor (small).
<tt> Mantém o mesmo espaçamento entre todos os caracteres. É utilizada para escrever o texto com fonte de máquina de escrever, chamada de monoespaçada. Por exemplo, a letra "i" ocupa o mesmo espaço horizontal que a letra "m".
<pre> Faz com que o navegador exiba o texto como ele é digitado, incluindo novas linhas, tabulações e espaços extras. Utilizada para texto pré-formatado.
<cite> Mantém o título de uma publicação em itálico.
<strong> Dá ênfase a uma palavra ou frase por meio do negrito.
<em> Dá ênfase a uma palavra ou frase por meio do itálico.
<blockquote> Permite a inclusão de uma citação longa ou de uma citação a um trecho de texto que tem origem em outro site.
<q> Atua de forma semelhante à tag blockquote, porém é aplicada para citações com uma única linha.

                <p>Este é um parágrafo</p>
                <strong>Texto em negrito</strong>

Listas

HTML permite criar listas ordenadas, listas desordenadas e listas de definições. Veja os exemplos:

Lista ordenada (numerada):

  • Cria uma lista que segue uma sequência numérica; Utiliza a tag <ol> (ordered list);
  • Cada item da lista é expresso entre a tag <li> (list item).

Exemplo:

  1. Item 1
  2. Item 2
  3. Item 3
  4. Item 4

Aqui está o código correspondente:

                <!DOCTYPE html>
                <html lang="pt-br">
                <head>
                    <meta charset="UTF-8">
                    <title>Listas</title>
                </head>
                <body> 
                <ol>
                    <li>Item 1</li>
                    <li>Item 2</li>
                    <li>Item 3</li>
                    <li>Item 4</li>
                </ol> 
                </body>
                </html>

Lista desordenada (marcada com pontos):

  • Cria uma lista que não segue uma sequência. Utiliza a tag <ul> (unordered list);
  • Cada item da lista é expresso entre a tag <li> (list item).

Exemplo:

  • Item 1
  • Item 2
  • Item 3
  • Item 4

Aqui está o código correspondente:

                <!DOCTYPE html>
                <html lang="pt-br">
                <head>
                    <meta charset="UTF-8">
                    <title>Listas</title>
                </head>
                <body> 
                <ul>
                    <li>Item 1</li>
                    <li>Item 2</li>
                    <li>Item 3</li>
                    <li>Item 4</li>
                </ul>  
                </body>
                </html>

Lista de definição:

  • Cria uma lista que define algum termo;
  • Utiliza a tag <dl> (definition list);
  • Um termo é representado pela tag <dt> (definition term);
  • A definição é representada pela tag <dd> (definition definition);
  • Uma <dt> pode ter uma ou mais <dd>.
              <!DOCTYPE html>
              <html>
                <head>
                  <meta charset="UTF-8" />
                  <title>Listas</title>
                </head>
                <body>
                  <p>Esta é uma lista de definição</p>
                  <dl>
                    <dt>Item 1</dt>
                    <dd>Definição 1</dd>
                    <dt>Item 2</dt>
                    <dd>Definição 2.1</dd>
                    <dd>Definição 2.2</dd>
                    <dt>Item 3</dt>
                    <dd>Definição 3</dd>
                  </dl>
                </body>
              </html>

Listas aninhadas:

  • Um item de uma lista pode se tornar uma nova lista;
  • Essa estrutura representa uma sub lista de uma lista principal.
                <!DOCTYPE html>
                <html>
                  <head>
                    <meta charset="UTF-8" />
                    <title>Listas</title>
                  </head>
                  <body>
                    <p>Esta é uma lista aninhada</p>
                    <ol>
                      <li>Item 1</li>
                      <ul>
                        <li>Subitem 1</li>
                        <li>Subitem 2</li>
                      </ul>
                      <li>Item 2</li>
                      <li>Item 3</li>
                    </ol>
                  </body>
                </html>

Links

  • Os links de navegação permitem que o usuário se mova entre diferentes seções da mesma página;
  • Utilizado com a tag <a>;
  • href: endereço da página na qual o link aponta;
  • mailto: abre a interface para envio de e-mail;
  • #id: uma parte específica da página;
  • Download: link é um download.
                <!DOCTYPE html>
                <html>
                  <head>
                    <meta charset="UTF-8" />
                    <title>Links</title>
                  </head>
                  <body>
                    <p id="paragrafo1">Estes são links</p>
                
                    <p>
                      Link para a página:
                      <a href="http://www.google.com.br">Google</a>
                    </p>
                
                    <p>
                      Link de envio de email:
                      <a href="mailto:juliana.silva@up.edu.br">
                        Enviar
                      </a>
                    </p>
                
                    <p>
                      Link para um local desta página:
                      <a href="#paragrafo1">Parágrafo</a>
                    </p>
                
                    <p>
                      Link para download:
                      <a href="download:exercicio1.txt">Baixar</a>
                    </p>
                  </body>
                </html>

Parâmetros de link:

  • target: especifica onde abrir o novo documento;
  • blank: abre em uma nova janela;
  • self: abre onde o documento atual está (default).
                <p>
                Link para a página:
                <a href="http://www.google.com.br" target="_blank">
                  Google
                </a>
              </p>

Imagens

Imagens são importantes para enriquecer o conteúdo visualmente e ilustrar informações. Para adicionar uma imagem, usamos a tag <img>:

            <img src="caminho-da-imagem.jpg" alt="Descrição da imagem" title = "Tag img"/>

O atributo src indica o caminho da imagem, o alt fornece uma descrição alternativa, essencial para acessibilidade e o title apresenta informação adicional sobre a imagem (tootip);

Adicionando imagens como links na página:

  • Você pode utilizar uma imagem como link;
  • Basta adicionar a tag dentro da tag <a>.
                <a href="http://www.google.com.br">
                  <img src="imagem.jpg" alt="Tag img">
                </a>

Atributo id

  • O atributo id serve para identificar unicamente um elemento de uma página;
  • Atributo global: qualquer tag do HTML pode possuir um id;
  • Seu valor deve começar um uma letra ou um underline;
  • Dois elementos em uma mesma página não podem possuir o mesmo id;
  • Auxilia o CSS e o JavaScript.
                <!DOCTYPE html>
                <html>
                    <head>
                        <meta charset="UTF-8"/>
                        <title>Imagens</title>
                    </head>
                    <body>
                        <p id="paragrafo1">Imagens na página</p>
                        <img 
                            src="imagen.jpg" 
                            alt="Uma imagem" 
                            title="Tag img"
                        />
                        <a href="http://www.google.com.br">
                            <img src="imagen.jpg" alt="Tag img"/>
                        </a>
                    </body>
                </html>

Atributo class

  • O atributo class serve para categorizar um grupo de elementos com o mesmo comportamento;
  • Atributo global: qualquer tag do HTML pode possuir um class;
  • Seu valor deve começar um uma letra ou um underline;
  • Auxilia o CSS e o JavaScript.
                <!DOCTYPE html>
                <html>
                    <head>
                        <meta charset="UTF-8"/>
                        <title>Atributos</title>
                    </head>
                    <body>
                        <p class="importante">atributo class</p>
                        <p>atributos categorizam elementos</p>
                        <p class="importante">atributo class</p>
                    </body>
                </html>

Elementos: Inline

Elementos Inline

  • Elementos que “sempre” são renderizados na mesma linha, ao término de um elemento anterior.

Elementos InLine em HTML

                <a> <abbr> <acronym> <b> <bdc> <big> <br>

Div

  • Permite o agrupamento de um conjunto de elementos em uma box block-level;
  • Cada conteúdo de uma div será apresentado em uma nova linha;
  • Esse comportamento pode ser alterado com CSS;
  • Melhora a organização do código;
  • É permitido colocar qualquer elemento dentro de uma div.
                <!DOCTYPE html>
                <html>
                  <head>
                    <meta charset="UTF-8" />
                    <title>Div</title>
                  </head>
                  <body>
                    <div>
                      <img src="imagem.jpg" />
                      <ul>
                        <li>Item 1</li>
                        <li>Item 2</li>
                      </ul>
                    </div>
                  </body>
                </html>

Seções

Aqui são definidas as seções do conteúdo da página, como "Início", "Sobre" e "Contato".

                <section id="home"> /* Seção Início */
                    <h2>Início</h2>
                    <p>Conteúdo principal da página de início</p>
                </section>
                <section id="sobre"> /* Seção Sobre */
                    <h2>Sobre Nós</h2>
                    <p>Informações sobre a empresa ou o projeto</p>
                </section>
                <section id="contato"> /* Seção Contato */
                    <h2>Contato</h2>
                    <p>Formulário de contato ou informações para entrar em contato</p>
                </section>

As seções são organizadas com a tag <section> e possuem um atributo id único para cada parte da página. Esses IDs são usados pelos links de navegação para facilitar a rolagem até as seções.

Tabelas

Tabelas organizam dados em linhas e colunas. Exemplo de uma tabela simples:

Tabela de Exemplos
Nome Idade Cidade
Ana 25 São Paulo
Bruno 30 Rio de Janeiro

O exemplo acima cria uma tabela com cabeçalho e duas linhas de dados. A tag <caption> fornece um título para a tabela.

Aqui está o código correspondente:

            <!DOCTYPE html>
                <html lang="pt-br">
                <head>
                    <meta charset="UTF-8">
                    <title>Tabelas</title>
                </head>
                <body> 
                <table>
                <caption>Tabela de Exemplos</caption>
                <thead>
                    <tr>
                        <th>Nome</th>
                        <th>Idade</th>
                        <th>Cidade</th>
                    </tr>
                </thead>
                <tbody>
                    <tr>
                        <td>Ana</td>
                        <td>25</td>
                        <td>São Paulo</td>
                    </tr>
                    <tr>
                        <td>Bruno</td>
                        <td>30</td>
                        <td>Rio de Janeiro</td>
                    </tr>
                </tbody>
                </table>
                </body>
                </html>

Formulários

Formulários são utilizados para a aquisição de informações do usuário.

  • Um formulário e englobado pela tag: <form method = “ ” action = “ ”>;
  • method define a forma de envio dos valores entre get ou post;
  • action aponta para a página no servidor que recebera as informações do formulário.

Abaixo está um exemplo de formulário de contato:

Respostas do Formulário:

Nome:

Email:

Assunto:

Mensagem:

Assim seria o código para criar esse conteúdo:

            <!DOCTYPE html>
                <html lang="pt-br">
                <head>
                    <meta charset="UTF-8">
                    <title>Formulários</title>
                </head>
                <body> 
                <form id="contactForm" action="*" method="post" onsubmit="return validateForm()">
                <label for="nome">Nome:</label>
                <input type="text" id="nome" name="nome" required>
                <span id="errorNome" class="error"></span>
            
                <label for="email">Email:</label>
                <input type="email" id="email" name="email" required>
                <span id="errorEmail" class="error"></span>
            
                <label for="assunto">Assunto:</label>
                <select id="assunto" name="assunto" required>
                    <option value="">Selecione um assunto</option>
                    <option value="duvida">Dúvida</option>
                    <option value="sugestao">Sugestão</option>
                    <option value="reclamacao">Reclamação</option>
                </select>
                <span id="errorAssunto" class="error"></span>
            
                <label for="mensagem">Mensagem:</label>
                <textarea id="mensagem" name="mensagem" rows="4" required></textarea>
                <span id="errorMensagem" class="error"></span>
            
                <button type="submit">Enviar</button>
                </form>
                </body>
                </html>
