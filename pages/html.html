HTML - Linguagem de Marcação de Hipertexto

  • (Hypertext Markup Language) é a linguagem padrão para criação de páginas na web.
    • Isto significa que é um código utilizado para descrever a estrutura de um documento, mas não é a verdadeira apresentação dele.
  • O HTML diz ao navegador, tudo sobre o conteúdo recebido, e a estrutura da página;
  • O HTML define cabeçalhos, parágrafos, listas, tabelas entre muitos outros elementos, por meio de marcas (tags).

A seguir, veremos a estrutura básica de um documento HTML:

Estrutura básica de um documento HTML

O exemplo abaixo mostra a estrutura básica de um documento HTML com suas principais tags:

                <!DOCTYPE html> /* Declara que o documento está utilizando HTML5 */
                <html lang="pt-br"> /* Tag raiz do documento HTML, com o idioma definido */
                <head> /* Cabeçalho do documento */
                    <meta charset="UTF-8"> /* Define a codificação de caracteres */
                    <title>Título da Página</title> /* Título exibido na aba do navegador */
                </head>
                <body> /* Corpo do documento, onde o conteúdo é exibido */
                    <h1>Cabeçalho Principal</h1> /* Cabeçalho ou título principal */
                    <p>Parágrafo de texto.</p> /* Texto em formato de parágrafo */
                </body>
                </html>

Neste exemplo, a tag <html> contém todo o conteúdo do documento. O <head> define metadados e informações sobre o documento, enquanto o <body> contém o conteúdo visível na página.

O que é uma Tag?

Tag é um comando usado em HTML para estruturar o conteúdo de uma página web. Ela é composta por uma tag de abertura e uma tag de fechamento. Entre essas tags, colocamos o conteúdo a ser exibido na página.

Tags HTML

Definições de Tags HTML
Tag HTML Descrição
<html> Define um documento HTML
<body> Define o corpo do documento
<h1> a <h6> Define título 1 a título 6
<p> Define um parágrafo
<br> Insere uma única quebra de linha
<hr> Define um filete horizontal
<!--> Define um comentário
<strong> Usado para aplicar negrito ao texto

Exemplo de Código:

<h1>Este é um título</h1>

Seções

Aqui são definidas as seções do conteúdo da página, como "Início", "Sobre" e "Contato".

                <section id="home"> /* Seção Início */
                    <h2>Início</h2>
                    <p>Conteúdo principal da página de início</p>
                </section>
                <section id="sobre"> /* Seção Sobre */
                    <h2>Sobre Nós</h2>
                    <p>Informações sobre a empresa ou o projeto</p>
                </section>
                <section id="contato"> /* Seção Contato */
                    <h2>Contato</h2>
                    <p>Formulário de contato ou informações para entrar em contato</p>
                </section>

As seções são organizadas com a tag <section> e possuem um atributo id único para cada parte da página. Esses IDs são usados pelos links de navegação para facilitar a rolagem até as seções.

Links de Navegação

Os links de navegação permitem que o usuário se mova entre diferentes seções da mesma página.

            <nav> /* Menu de navegação */
                <ul>
                    <li><a href="#inicio">Início</a></li>
                    <li><a href="#sobre">Sobre</a></li>
                    <li><a href="#contato">Contato</a></li>
                </ul>
            </nav>
            <section id="inicio"> /* Seção Início */
                <h2>Início</h2>
                <p>Conteúdo da seção de início</p>
            </section>
            <section id="sobre"> /* Seção Sobre */
                <h2>Sobre Nós</h2>
                <p>Conteúdo da seção sobre</p>
            </section>
            <section id="contato"> /* Seção Contato */
                <h2>Contato</h2>
                <p>Conteúdo da seção de contato</p>
            </section>

O exemplo acima mostra um menu de navegação que leva o usuário para as seções "Início", "Sobre" e "Contato" dentro da mesma página. Cada link tem um atributo href que aponta para o ID de uma seção específica.

Listas

HTML permite criar listas ordenadas, listas desordenadas e listas de definições. Veja os exemplos:

Lista desordenada (marcada com pontos):

  • Item 1
  • Item 2
  • Item 3
  • Item 4

Aqui está o código correspondente:

                <!DOCTYPE html>
                <html lang="pt-br">
                <head>
                    <meta charset="UTF-8">
                    <title>Listas</title>
                </head>
                <body> 
                <ul>
                    <li>Item 1</li>
                    <li>Item 2</li>
                    <li>Item 3</li>
                    <li>Item 4</li>
                </ul>  
                </body>
                </html>

Lista ordenada (numerada):

  1. Item 1
  2. Item 2
  3. Item 3
  4. Item 4

Aqui está o código correspondente:

                <!DOCTYPE html>
                <html lang="pt-br">
                <head>
                    <meta charset="UTF-8">
                    <title>Listas</title>
                </head>
                <body> 
                <ol>
                    <li>Item 1</li>
                    <li>Item 2</li>
                    <li>Item 3</li>
                    <li>Item 4</li>
                </ol> 
                </body>
                </html>

Imagens

Imagens são importantes para enriquecer o conteúdo visualmente e ilustrar informações. Para adicionar uma imagem, usamos a tag <img>:

            <img src="caminho-da-imagem.jpg" alt="Descrição da imagem" width="300" height="200">

O atributo src indica o caminho da imagem, e o alt fornece uma descrição alternativa, essencial para acessibilidade.

Tabelas

Tabelas organizam dados em linhas e colunas. Exemplo de uma tabela simples:

Tabela de Exemplos
Nome Idade Cidade
Ana 25 São Paulo
Bruno 30 Rio de Janeiro

O exemplo acima cria uma tabela com cabeçalho e duas linhas de dados. A tag <caption> fornece um título para a tabela.

Aqui está o código correspondente:

 
            <!DOCTYPE html>
                <html lang="pt-br">
                <head>
                    <meta charset="UTF-8">
                    <title>Tabelas</title>
                </head>
                <body> 
                <table>
                <caption>Tabela de Exemplos</caption>
                <thead>
                    <tr>
                        <th>Nome</th>
                        <th>Idade</th>
                        <th>Cidade</th>
                    </tr>
                </thead>
                <tbody>
                    <tr>
                        <td>Ana</td>
                        <td>25</td>
                        <td>São Paulo</td>
                    </tr>
                    <tr>
                        <td>Bruno</td>
                        <td>30</td>
                        <td>Rio de Janeiro</td>
                    </tr>
                </tbody>
                </table>
                </body>
                </html>

Formulários

Formulários coletam dados dos usuários. Abaixo está um exemplo de formulário de contato:

Assim seria o código para criar esse conteúdo:

            <!DOCTYPE html>
                <html lang="pt-br">
                <head>
                    <meta charset="UTF-8">
                    <title>Formulários</title>
                </head>
                <body> 
                <form id="contactForm" action="*" method="post" onsubmit="return validateForm()">
                <label for="nome">Nome:</label>
                <input type="text" id="nome" name="nome" required>
                <span id="errorNome" class="error"></span>
            
                <label for="email">Email:</label>
                <input type="email" id="email" name="email" required>
                <span id="errorEmail" class="error"></span>
            
                <label for="assunto">Assunto:</label>
                <select id="assunto" name="assunto" required>
                    <option value="">Selecione um assunto</option>
                    <option value="duvida">Dúvida</option>
                    <option value="sugestao">Sugestão</option>
                    <option value="reclamacao">Reclamação</option>
                </select>
                <span id="errorAssunto" class="error"></span>
            
                <label for="mensagem">Mensagem:</label>
                <textarea id="mensagem" name="mensagem" rows="4" required></textarea>
                <span id="errorMensagem" class="error"></span>
            
                <button type="submit">Enviar</button>
                </form>
                </body>
                </html>
