HTML - Linguagem de Marcação de Hipertexto

  • (Hypertext Markup Language) é a linguagem padrão para criação de páginas na web.
    • Isto significa que é um código utilizado para descrever a estrutura de um documento, mas não é a verdadeira apresentação dele.
  • O HTML diz ao navegador, tudo sobre o conteúdo recebido, e a estrutura da página;
  • O HTML define cabeçalhos, parágrafos, listas, tabelas entre muitos outros elementos, por meio de marcas (tags).

A seguir, veremos a estrutura básica de um documento HTML:

Estrutura básica de um documento HTML

O exemplo abaixo mostra a estrutura básica de um documento HTML com suas principais tags:

                <!DOCTYPE html> /* Declara que o documento está utilizando HTML5 */
                <html lang="pt-br"> /* Tag raiz do documento HTML, com o idioma definido */
                <head> /* Cabeçalho do documento */
                    <meta charset="UTF-8"> /* Define a codificação de caracteres */
                    <title>Título da Página</title> /* Título exibido na aba do navegador */
                </head>
                <body> /* Corpo do documento, onde o conteúdo é exibido */
                    <h1>Cabeçalho Principal</h1> /* Cabeçalho ou título principal */
                    <p>Parágrafo de texto.</p> /* Texto em formato de parágrafo */
                </body>
                </html>

Neste exemplo, a tag <html> contém todo o conteúdo do documento. O <head> define metadados e informações sobre o documento, enquanto o <body> contém o conteúdo visível na página.

Listas

HTML permite criar listas ordenadas e desordenadas. Veja os exemplos:

Lista desordenada (marcada com pontos):

  • Item 1
  • Item 2

Lista ordenada (numerada):

  1. Item 1
  2. Item 2

Imagens

Imagens são importantes para enriquecer o conteúdo visualmente e ilustrar informações. Para adicionar uma imagem, usamos a tag <img>:

            <img src="caminho-da-imagem.jpg" alt="Descrição da imagem" width="300" height="200">

O atributo src indica o caminho da imagem, e o alt fornece uma descrição alternativa, essencial para acessibilidade.
