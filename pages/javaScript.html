JavaScript - Linguagem de Programação para Web

JavaScript é uma linguagem que permite adicionar interatividade às páginas web. Com JavaScript, você pode criar efeitos dinâmicos, validar formulários e muito mais.

Exemplo de Código JavaScript

JavaScript permite adicionar interatividade às páginas web. O exemplo a seguir demonstra como exibir uma saudação ao clicar em um botão.

                document.addEventListener('DOMContentLoaded', () => {
   
                    const button = document.getElementById('saudacao');
                
                    if (button) {
                        button.addEventListener('click', () => {
                            alert('Bem-vindo à nossa apostila web!');
                        });
                    }
                });

Este código utiliza um evento chamado `DOMContentLoaded`, que garante que o script será executado somente após todo o HTML ter sido carregado.

Métodos HTTP - Interação com APIs

Os métodos HTTP são utilizados para realizar operações de envio e recebimento de dados entre o navegador e o servidor. Os principais métodos são:

  • GET: Solicita dados de um servidor sem alterar o estado do recurso. Usado para carregar informações, como ao acessar uma página.
  • POST: Envia dados ao servidor para criar um novo recurso. Frequentemente utilizado em formulários para enviar informações de cadastro, por exemplo.
  • PUT: Atualiza dados já existentes no servidor. É utilizado para modificar recursos de forma completa, enviando os dados atualizados no corpo da requisição.
  • DELETE: Remove dados do servidor. Esse método é restrito e precisa ser usado com cuidado para evitar exclusões acidentais.

Esses métodos são essenciais para a comunicação entre cliente e servidor, principalmente em aplicações web que consomem APIs.

Exemplo de Requisição HTTP com JavaScript

Para realizar requisições HTTP em JavaScript, você pode utilizar a API Fetch, que permite enviar e receber dados de servidores remotos. O exemplo a seguir demonstra como fazer uma requisição GET para a API do GitHub:

                document.addEventListener('DOMContentLoaded', () => {
   
                    fetch('https://api.github.com/users/octocat')
                        .then(response => response.json())
                        .then(data => {
                            console.log(data);
                        })
                        .catch(error => {
                            console.error('Erro ao carregar dados:', error);
                        });
                });

Este código utiliza o método `fetch` para realizar uma requisição GET à API do GitHub, que retorna informações sobre o usuário `octocat`. O resultado é tratado com as funções `then` e `catch`, que lidam com os dados recebidos e possíveis erros na requisição.
