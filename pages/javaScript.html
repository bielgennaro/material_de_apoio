JavaScript - Linguagem de Programação para Web

JavaScript é uma linguagem que permite adicionar interatividade às páginas web. Com JavaScript, você pode criar efeitos dinâmicos, validar formulários e muito mais.

Exemplo de Código JavaScript

JavaScript permite adicionar interatividade às páginas web. O exemplo a seguir demonstra como exibir uma saudação ao clicar em um botão. 

document.addEventListener('DOMContentLoaded', () => {
    const buttonSaudacao = document.getElementById('saudacao');
    if (buttonSaudacao) {
        buttonSaudacao.addEventListener('click', () => {
            const hours = new Date().getHours();
            const greetings = hours < 12 ? 'Bom dia' : hours < 18 ? 'Boa tarde' : 'Boa noite';
            alert(`${greetings}! Bem-vindo à Apostila Web!`);
        });
    }
});

Com Javascript podemos manipular a DOM de várias maneiras, nesse exemplo utilizamos a funcão getHours para recolher as horas do navegador podendo verificar a hora que o usuário se encontra, sendo assim, podemos manipular de forma dinâmica se ele receberá um "Bom dia" ou "Boa noite" dependendo do seu horário.

Métodos HTTP - Interação com APIs

Os métodos HTTP são utilizados para realizar operações de envio e recebimento de dados entre o navegador e o servidor. Os principais métodos são:

  • GET: Solicita dados de um servidor sem alterar o estado do recurso. Usado para carregar informações, como ao acessar uma página.
  • POST: Envia dados ao servidor para criar um novo recurso. Frequentemente utilizado em formulários para enviar informações de cadastro, por exemplo.
  • PUT: Atualiza dados já existentes no servidor. É utilizado para modificar recursos de forma completa, enviando os dados atualizados no corpo da requisição.
  • DELETE: Remove dados do servidor. Esse método é restrito e precisa ser usado com cuidado para evitar exclusões acidentais.

Esses métodos são essenciais para a comunicação entre cliente e servidor, principalmente em aplicações web que consomem APIs.

Exemplo de Requisição HTTP com JavaScript

Para realizar requisições HTTP em JavaScript, você pode utilizar a API Fetch, que permite enviar e receber dados de servidores remotos. O exemplo a seguir demonstra como fazer uma requisição GET para a API do GitHub: 

document.addEventListener('DOMContentLoaded', () => {
    fetch('https://api.github.com/users/octocat')
        .then(response => response.json())
        .then(data => {
            console.log(data);
        })
        .catch(error => {
            console.error('Erro ao carregar dados:', error);
        });
});

Este código utiliza o método `fetch` para realizar uma requisição GET à API do GitHub, que retorna informações sobre o usuário `octocat`. O resultado é tratado com as funções `then` e `catch`, que lidam com os dados recebidos e possíveis erros na requisição.

Elementos Dinâmicos

Uma situação muito comum ao utilizar Javascript é adicionar itens no HTML por exemplo, como uma lista de compras. Abaixo, temos um exemplo de como adicionar itens a uma lista dinamicamente.

Adicione novos itens à lista:

    Código do exemplo de lista dinâmica

    document.addEventListener('DOMContentLoaded', () => {
    const addItemButton = document.getElementById('addItemButton');
    const dynamicList = document.getElementById('dynamicList');

    if (addItemButton) {
        addItemButton.addEventListener('click', () => {
            const newItem = document.createElement('li');
            newItem.textContent = `Item ${dynamicList.children.length + 1}`;
            dynamicList.appendChild(newItem);
        });
    }
});

    Variáveis

    Variáveis são usadas para armazenar dados que podem ser reutilizados ao longo do código. Em JavaScript, existem três palavras-chave principais para declarar variáveis:

    • var: Escopo global ou local (não recomendada).
    • let: Escopo de bloco.
    • const: Não permite reatribuição.
    const nome = 'João';
let idade = 25;
idade = 26;
var profissao = 'Desenvolvedor';

    Tipos de Dados

    Em JavaScript, os principais tipos de dados são:

    • String: Texto delimitado por aspas ('texto', "texto", ou `texto`).
    • Number: Números inteiros ou decimais (42, 3.14).
    • Boolean: Verdadeiro ou falso (true, false).
    • Array: Lista de valores ([1, 2, 3]).
    • Object: Coleção de pares chave-valor ({ chave: 'valor' }).
    • null: Valor vazio ou nulo.
    • undefined: Variável declarada sem valor atribuído. 
    const texto = "Olá, mundo!";
const numero = 42;
const verdadeiro = true;
const lista = [1, 2, 3, 4];
const objeto = { nome: "Ana", idade: 30 };

    Condicionais

    Condicionais são usadas para executar diferentes blocos de código com base em uma condição. 

    const hora = 14;

if (hora < 12) {
    console.log('Bom dia!');
} else if (hora < 18) {
    console.log('Boa tarde!');
} else {
    console.log('Boa noite!');
}

// Operador ternário
const saudacao = hora < 12 ? 'Bom dia!' : 'Boa tarde ou noite!';
console.log(saudacao);

    Loops

    Loops permitem executar um bloco de código repetidamente.

    Exemplo: loop for

    for (let i = 0; i < 5; i++) {
    console.log(`Número: ${i}`);
}

    Exemplo: loop while

    let contador = 0;
while (contador < 5) {
    console.log(`Contador: ${contador}`);
    contador++;
}

    Funções

    Funções são blocos de código reutilizáveis. Elas podem receber parâmetros e retornar valores.

    Função tradicional

    function saudacao(nome) {
    return `Olá, ${nome}!`;
}
console.log(saudacao('Maria'));

    Arrow Function

    const saudacao = nome => `Olá, ${nome}!`;
console.log(saudacao('Carlos'));

    Manipulação da DOM

    A DOM (Document Object Model) representa a estrutura de uma página web. Com JavaScript, podemos modificar elementos, estilos e eventos.

    Exemplo de Modificação de Texto

    document.addEventListener('DOMContentLoaded', () => {
    const titulo = document.getElementById('titulo');
    if (titulo) {
        titulo.textContent = 'Título Alterado com JavaScript!';
    }
});

    Exemplo de Alteração de Estilos

    document.addEventListener('DOMContentLoaded', () => {
    const elemento = document.getElementById('estilo');
    if (elemento) {
        elemento.style.color = 'blue';
        elemento.style.fontSize = '20px';
    }
});

    Eventos

    Eventos permitem interatividade ao capturar ações do usuário, como cliques ou teclas pressionadas. 

    document.addEventListener('DOMContentLoaded', () => {
    const botao = document.getElementById('meuBotao');
    if (botao) {
        botao.addEventListener('click', () => {
            alert('Botão clicado!');
        });
    }
});
